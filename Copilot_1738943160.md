## GitHub Copilot拡張：Agent ModeとCopilot EditsがVS Codeで利用可能に

### 1. はじめに

本レポートでは、heise onlineの記事「GitHub Copilot extended: Agent Mode and Copilot Edits available in VS Code」に基づき、GitHub Copilotの最新機能拡張であるAgent ModeとCopilot Editsについて、その概要、機能、および開発者への影響を客観的に分析する。

### 2. GitHub Copilotの概要

GitHub Copilotは、GitHubとOpenAIが共同で開発したAIペアプログラマーである。開発者がコードを書く際に、リアルタイムでコードの提案や補完を行うことで、開発効率の向上を支援する。

### 3. Agent Mode：自律的なコードレビューと修正

#### 3.1 Agent Modeの概要

Agent Modeは、GitHub Copilotの新しい機能拡張であり、コードの自動レビュー、エラーの特定と修正を自律的に行う。ターミナルコマンドの提案やランタイムエラーの分析も可能にする。

#### 3.2 Agent Modeの機能

* **コードレビューの自動化：** コードの品質を向上させるために、潜在的な問題点や改善点を自動的に検出する。
* **エラーの特定と修正：** コード内のエラーを特定し、修正案を提示することで、デバッグ作業を効率化する。
* **ターミナルコマンドの提案：** 開発者がタスクを実行するために必要なターミナルコマンドを提案する。
* **ランタイムエラーの分析：** 実行時に発生したエラーの原因を分析し、解決策を提案する。

#### 3.3 Agent Modeの現状と今後の展望

Agent Modeは現在プレビュー版であり、今後、サポートされるすべての開発環境に拡張される予定である。

### 4. Copilot Edits：自然言語による複数ファイル編集

#### 4.1 Copilot Editsの概要

Copilot Editsは、Visual Studio Code（VS Code）で利用可能な機能拡張であり、自然言語を使用して複数のファイルにわたる変更を可能にする。

#### 4.2 Copilot Editsの機能

* **自然言語による編集：** 開発者は、自然言語で編集内容を指示することで、コードを修正できる。
* **複数ファイル編集：** 複数のファイルにまたがる変更を一度に行うことができる。
* **デュアルモデルアーキテクチャ：** 効率的かつ正確な編集提案のために、デュアルモデルアーキテクチャを採用している。
* **言語モデルの選択：** OpenAIのGPT-4oやGoogleのGemini 2.0 Flashなど、異なる言語モデルを選択できる。

#### 4.3 Copilot Editsの現状と今後の展望

Copilot EditsはVS Codeで利用可能であり、Visual Studio 2022のプレビュー版としても提供されている。パフォーマンスを最適化するための改善が計画されている。

### 5. Project Padawan：ルーチンタスクの自動化

#### 5.1 Project Padawanの概要

Project Padawanは、ソフトウェア開発者を支援するための自律エージェントを統合するプロジェクトである。

#### 5.2 Project Padawanの目的

ルーチンタスクを自動化することで、開発者がより複雑な作業に集中できるようにすることを目的とする。

#### 5.3 Project Padawanの機能

* **ルーチンタスクの自動化：** コード生成、テスト、デプロイメントなどのルーチンタスクを自動化する。
* **GitHub内での作業：** GitHub内で直接タスクを実行し、開発プロセス全体を処理する。

### 6. 開発者への影響

GitHub Copilotの拡張機能（Agent Mode、Copilot Edits、Project Padawan）は、開発者の生産性向上、コード品質の向上、および開発プロセスの効率化に貢献する可能性がある。

### 7. まとめ

GitHub CopilotのAgent ModeとCopilot Editsは、AIを活用した開発支援ツールとして、開発者のワークフローに大きな変革をもたらす可能性がある。今後の機能拡張や対応環境の拡大に注目が集まる。
