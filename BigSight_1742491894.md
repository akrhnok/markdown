# AnimeJapan 2025：サウジアラビアのマンガプロダクションズ出展とビジネスセミナー開催

## 1. はじめに

本レポートは、マンガプロダクションズ株式会社が発表したプレスリリースに基づき、世界最大級のアニメイベント「AnimeJapan 2025」への同社の出展内容と、開催されるビジネスセミナーについて客観的に分析するものである。

## 2. AnimeJapan 2025におけるマンガプロダクションズの出展概要

### 2.1 出展目的

マンガプロダクションズは、AnimeJapan 2025への出展を通じて、日本のアニメファンへのアピールと、中東地域におけるアニメビジネスの展開を加速させることを目指していると考えられる。

### 2.2 パブリックデイブース

パブリックデイ（一般公開日）では、来場者向けの様々なイベントが企画されている。

* **QRコードスタンプラリー:** ブース内に隠されたQRコードを集めることで、プレゼントがもらえる。
* **カリグラフィーサービス:** アラビア語のカリグラフィーで来場者の名前を手書きするサービス。
* **なりきりサイネージ:** サウジアラビアの衣装を着た姿をサイネージを通して体験できる。
* **伝統衣装コスプレ体験:** サウジアラビアの伝統的な衣装（トーブ、アバヤ）のコスプレ体験。
* **ゲーム試遊:** 「アサティール 冒険の物語」や「真・三國無双 ORIGINS」などのゲーム試遊。
* **サウジ・ラウンジ:** サウジアラビアのテントとリビングを再現し、デーツ（ナツメヤシ）とマムール（クッキー）を提供するラウンジ。

### 2.3 ブースデザイン

ブースは、サウジアラビアの伝統的な家と居間をイメージしたデザインとなる予定である。

## 3. ビジネスセミナー

### 3.1 開催概要

* **日時:** 2025年3月24日（月）17:30～18:00
* **場所:** 東京ビッグサイト 会議棟1F
* **講演者:** マンガプロダクションズCEO ブカーリ イサム
* **テーマ:** 「アニメ新時代！日本×サウジが仕掛ける「サウジニメーション」の未来」

### 3.2 セミナー内容

中東地域におけるエンターテインメント、イベント、アニメビジネスの現状と将来展望について、CEOのブカーリ イサム氏が講演を行う。中東地域のアニメ市場の成長性、マンガプロダクションズの役割、日本企業へのビジネスチャンスなどが語られると推測される。

### 3.3 登壇者プロフィール

ブカーリ イサム氏は、マンガプロダクションズのCEOであり、東京大学ムハンマド・ビン・サルマン未来科学技術センター実行委員会執行委員、マンガアラビア編集長などを務める。日本とサウジアラビアの文化交流に貢献し、日本の外務大臣賞を2度受賞している。

## 4. AnimeJapan 2025の概要

AnimeJapan 2025は、アニメに関する展示、物販、イベント、声優イベント、ライブ、トークショー、コスプレイベントなど、アニメに関連するあらゆる要素が集結する総合イベントである。

* **パブリックデイ:** 2025年3月22日（土）・23日（日）
* **ビジネスデイ:** 2025年3月24日（月）・25日（火）

## 5. マンガプロダクションズについて

マンガプロダクションズは、中東のアラブ諸国を代表するIPマネジメント、アニメ・ゲーム・マンガの制作・配給会社である。日本のコンテンツを世界に発信する活動を行っている。

## 6. まとめ

マンガプロダクションズのAnimeJapan 2025への出展は、中東地域におけるアニメビジネスの拡大を目指す同社の戦略の一環である。パブリックデイでの多様なイベントと、ビジネスデイでのセミナーを通じて、日本のアニメファンとビジネス関係者へのアピールを図る。中東地域のアニメ市場の成長性と、マンガプロダクションズの役割に注目が集まる。
